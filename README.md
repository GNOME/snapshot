## Requirements

Snapshot requires:

- PipeWire
- WirePlumber
- gstreamer-pipewire plugin
- gst-plugin-gtk4 plugin
- gst-plugins-bad plugin
- Optional: libcamera
- Optional: gstreamer-libcamera plugin (if using libcamera)
